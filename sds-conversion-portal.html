Usage guide

How to use the SDS conversion portal.

Workflow overview
  1. Author the SDS in the software using the closest available source jurisdiction.
  2. Export the SDS as PDF from the authoring software.
  3. Open a new conversion request and complete all four steps.
  4. Upload: the authored SDS, the target jurisdiction template, and your section 8/15 data exports.
  5. Submit. Claude will convert, populate section 8/15, and QC the SDS.
  6. Download the output DOCX and complete the manual steps (section 1, logo, revision notes).
File format tips
  • Authored SDS: PDF preferred; DOCX accepted.
  • Template: DOCX preferred for best output formatting; PDF accepted for reference.
  • Section 8/15 data: XLSX or CSV from your data provider. Multi-substance exports are fine — enter CAS numbers in the Notes field to guide extraction.
  • Max file size: 25 MB per file.
What Claude does — and doesn't — do

Claude will:

  • Restructure the SDS content into the target jurisdiction template
  • Extract OELs, BEIs, and regulatory listings from your uploaded data files
  • Align classification language, units, and GHS elements to the target jurisdiction
  • Flag QC issues and items requiring manual completion

Claude will not (manual steps):

  • Fill in section 1 supplier / emergency contact info
  • Insert the customer logo
  • Set the SDS revision date or revision notes
